# Vector Animation
A project using [TwoEasyEngine](https://github.com/niiicolai/two-easy-engine) to introduce the foundational concepts of vector mathematics through 2D animations.

## Development

### Install dependencies

```bash
npm install
```

### Build static files

```bash
npm run build
```

### Development

```bash
npm run dev
```
